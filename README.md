### 프로젝트 소개

이 프로젝트는 내배캠 Todo 게시판의 백엔드를 담당합니다.

게시글의 생성, 전체조회, 단건 조회, 수정, 삭제를 구현했습니다.

댓글의 생성, 수정, 삭제를 구현했습니다.

회원가입 및 로그인을 구현했습니다.

게시물의 좋아요 및 취소 기능을 구현했습니다.
